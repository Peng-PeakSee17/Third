# Third
A paper recycling bin specially designed for Brother Ke

## 一、 核心含义

#### 		”所有关系之外的观察者聚集地“

## 二、Slogan

- [ ] **“站在局外，看清一切”	——	"Stand outside the situation and see everything clearly"** 
- [ ] **“不是当事人，但比当事人更清醒”   ——    "Not the party involved, but more clear-headed than the party involved"** *情绪*
- [ ] **”万物的第三种观点“    ——    "The Third View of Everything"**
- [ ] **“不参与，但不缺席”    ——     "Not participating, but not absent"**
